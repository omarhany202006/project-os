# Contributing to Kernel Panic Logger

Thank you for your interest in contributing! This project welcomes contributions to improve kernel panic handling for xv6 and other operating systems.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Code Style

- Use C99 standard
- Follow POSIX conventions
- Include proper error checking
- Add comments for complex logic
- Test on both Windows and Linux environments

## Building and Testing

```bash
# On Windows
./build.bat

# On Linux/macOS
./build.sh
```

## Reporting Issues

Please use the GitHub Issues tracker with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Platform information (Windows/Linux/etc)
